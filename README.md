# 🌍 Wanderlust – Airbnb Clone

Wanderlust is a full-stack web application inspired by Airbnb, where users can create, view, edit, and delete property listings.

---

## 🚀 Features

- 🏠 Create new property listings
- ✏ Edit existing listings
- 🗑 Delete listings
- 📄 View detailed listing pages
- 🌱 Seed database with sample data
- 🗂 RESTful routing architecture
- 🎨 EJS templating with Bootstrap styling

---

## 🛠 Tech Stack

### 🔹 Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### 🔹 Frontend
- EJS (Embedded JavaScript)
- Bootstrap
- CSS
